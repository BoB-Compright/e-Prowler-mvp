# Project Guide

## Agent skills

### Issue tracker

GitHub Issues에서 이슈를 관리합니다. 외부 PR은 일반 컨트리뷰션으로만 처리합니다. 자세한 내용은 `docs/agents/issue-tracker.md` 참고.

### Triage labels

기본 레이블 어휘를 사용합니다: `needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`. 자세한 내용은 `docs/agents/triage-labels.md` 참고.

### Domain docs

단일 컨텍스트: `CONTEXT.md` + `docs/adr/`이 리포 루트에 있습니다. 자세한 내용은 `docs/agents/domain.md` 참고.
